# 02 PRD

## 产品定位

AI 智能工牌生成系统是一个面向企业 HR / 行政场景的内部提效工具，用 AI 处理头像标准化，用模板系统保证工牌输出一致。

## 目标用户

- HR：负责员工入职资料和工牌制作。
- 行政：负责公司证件、访客证或临时工牌。
- 小团队负责人：需要快速生成统一视觉的团队证件。

## 核心流程

```text
进入上传页
→ 上传员工照片
→ 填写员工信息
→ 后端生成证件照
→ 后端合成工牌 PNG
→ 前端展示结果
→ 用户下载图片
```

## 功能范围

| 模块 | 功能 | 优先级 |
| --- | --- | --- |
| 上传表单 | 上传照片、填写员工基础信息 | P0 |
| AI 证件照 | 调用 HivisionIDPhotos 输出标准头像 | P0 |
| 工牌合成 | 将头像和字段写入 HTML/CSS 模板 | P0 |
| 结果页 | 展示证件照和工牌图，支持下载 | P0 |
| 管理页 | 展示生成任务和状态 | P1 |
| 模板切换 | 支持不同企业模板 | P1 |
| 批量生成 | 批量上传员工信息和照片 | P2 |

## 验收标准

- 用户能完成从上传照片到下载工牌 PNG 的完整流程。
- 生成结果包含头像、姓名、部门、岗位、工号等字段。
- 照片处理失败时，前端能看到明确错误提示。
- Demo 不依赖真实员工数据即可演示。
