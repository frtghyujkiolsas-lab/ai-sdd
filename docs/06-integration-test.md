# 06 前后端联调

## 联调目标

确认前端表单、后端接口、AI 证件照处理和工牌 PNG 输出在同一条链路中可以稳定跑通。

## 联调路径

```text
前端上传真实测试图
→ 调用 `/api/generate`
→ 后端处理照片和模板
→ 前端展示 `id_photo` 与 `badge_png`
→ 用户下载最终 PNG
```

## 测试用例

| 场景 | 预期 |
| --- | --- |
| 正常照片 + 完整字段 | 成功生成证件照和工牌 PNG |
| 缺少姓名 | 前端阻止提交或后端返回字段错误 |
| 未上传照片 | 返回照片必填提示 |
| 照片不清晰 | 返回处理失败提示 |
| HivisionIDPhotos 未启动 | 返回服务不可用提示 |

## 联调检查点

- 前端字段名与 API 契约一致。
- 后端返回字段与结果页读取字段一致。
- 生成结果可正常下载。
- 错误提示对非技术用户可理解。
- 本地构建和 Python 语法检查通过。
