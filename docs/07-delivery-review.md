## 验收口径

这个项目的验收重点不是“是否适合直接进企业生产”，而是是否证明了完整产品闭环：

```text
需求明确
→ 原型可看
→ 接口可调
→ 后端可跑
→ 前后端可联调
→ 结果可交付
```

## 可继续迭代方向

- 接入更多工牌模板。
- 增加批量导入员工信息。
- 增加任务历史和重新生成。
- 增加模板配置后台。
- 增加 Docker Compose 一键启动。

## 隐私边界

公开仓库只放匿名样例和测试数据，不提交真实员工照片、真实工号、真实公司内部资料或生成日志。
